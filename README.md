# Stenography Learning Optimization & Data Analysis Project

## Overview

This project applies data analysis and language frequency optimization to improve the process of learning stenography (steno). The goal was to identify the highest-value concepts, words, and multi-word phrases to study first in order to maximize real-world writing speed improvements.

Instead of treating all steno briefs equally, this project analyzes language patterns to determine which shortcuts provide the greatest return on learning time.

## Data Analysis & Optimization

### Multi-Word Brief Frequency Analysis

A dataset of the **10,000 most frequent multi-word phrases in English** was analyzed to identify the most valuable one-stroke steno briefs to learn.

The analysis focused on:

* Which phrases appear most frequently in everyday language
* How many briefs provide meaningful coverage of common communication
* How learning priority can be optimized based on usage frequency

Findings:

* The first **5,000 high-frequency briefs provide approximately 85% coverage** of the analyzed phrase frequency.
* The highest-frequency briefs provide the greatest improvement in writing efficiency, demonstrating that targeted memorization is more effective than learning shortcuts randomly.

## Stroke Efficiency Analysis

The project analyzed the relationship between:

* Strokes per word
* Strokes per second
* Words per minute (WPM)

Example:

* At 3 strokes per second:

  * 1 stroke per word ≈ 180 WPM
  * 1.4 strokes per word ≈ 130 WPM

This demonstrates how reducing average strokes per word directly increases maximum writing speed.

## Stenography Theory Optimization

The project explored the tradeoff between different steno theories:

* **Long theories:** Easier to learn but require more strokes per word.
* **Short theories:** Require more memorization but reduce strokes and improve long-term speed potential.

Example:

The word **"California"** can be written:

* Using multiple strokes by separating syllables (long theory)
* Using a single stroke brief (short theory)

The analysis shows that minimizing strokes per word is a key factor in achieving higher writing speeds.

## Learning Strategy Development

Using frequency analysis and spaced repetition principles, a structured learning workflow was developed:

1. Learn high-frequency briefs using flashcards.
2. Practice the same words through speed-building drills.
3. Review using spaced repetition algorithms.
4. Prioritize commonly used language patterns over random vocabulary.

This approach applies data-driven learning optimization to reduce study time while improving retention.
