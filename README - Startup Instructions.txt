Learn the briefs using Anki, and drill them until it's second nature.
Important: Every brief should be written in a single stroke, even if a flashcard has more than one word.
Example: "of the" written as -FT

Resources:
Anki Software: https://apps.ankiweb.net/
(optional) Plover Software: https://www.openstenoproject.org/plover/

When opening the flashcard deck into Anki:
- Select "Import any deck presets"
- Ensure FSRS is turned on in the deck settings, and inside all subdecks (gear icons)
- Every so often (e.g. every couple weeks), go back into the deck settings, and press the "Optimize" button to ensure the cards are shown to you at the optimal time.

When learning new briefs for the first time:
- Use the included brief blueprints with corresponding papertape entries for each brief, if a new dictionary entry is required.
- Provided .JSON files can be added into the plover software and searched through using the 'lookup' tool. This can be useful when looking for preexisting entries to work off of. Otherwise, use CTRL+F to search through the file.

When studying flashcards on Anki:
- Attempt to use steno to write into the text box at the front of the flashcard. If the brief is correct, press "Good" on the card to increase its interval size until you see it next time.
- Spaced repetition study forces you to retrieve information which strengthens memory and ensures long term retention, which then can be drilled into muscle memory with the included lessons.

When speedbuilding the briefs on lesson drills:
- Refresh page to rescramble
- Drill until comfortable, then move on to the next lesson
